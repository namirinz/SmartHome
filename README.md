# SmartHome
This smart home project has built for OCOP 2019 at Triamudomsuksa Pattanakarn School.

# Equipemnt
1) Raspberry Pi 3B+
2) Raspberry Pi camera v2
3) Servo motor
4) DHT 22
5) Micro Switch
6) RGB LED

# Feature
1) Face Recognition
2) Raspi x Line Notify
3) Control Panel
